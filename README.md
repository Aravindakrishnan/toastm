# What is Toastm ? 🤔

    It is Just a Package for Toast / Notification Message 📜

# Installation ⚙️

    ```
        npm install toastm
    ```

# How to Use toastm 🤔

    ```javascript

    import { addToastm } from "toastm";

    addToastm({
        color: "primary",
        position:"center",
        text:"Toastm notification box 😄"
    })

    ```

## Options 📜

    * *color* - primary | secondary | success | danger | warning | dark | special
    * *positon* - left | center | right
    * *text* - anything
